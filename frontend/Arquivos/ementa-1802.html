Ementa:

O Arquivo Intermediário como eixo de destinação. Os sistemas arquivísticos sob esta ótica. Níveis de sistema. A administração da eliminação. O recolhimento aos arquivos permanentes.

Programa:

Módulo 1 - O Problema - As massas documentais acumuladas

a - definição;

b - constituição;

c - diagnóstico da situação arquivística brasileira;

d - relação com os arquivos montados nos setores de trabalho.



Módulo 2 - Arquivo intermediário: aspectos conceituais

a - histórico;

b - o ciclo da informação arquivística: o conceito das três idades;

c - a construção da definição do conceito de arquivo intermediário;

d - a questão do acesso nos arquivos intermediários.



Módulo 3 - O Arquivo Intermediário e os Sistemas Arquivísticos

a - os níveis de sistema;

b - a estrutura;

c - o funcionamento.



Módulo 4 - A legislação arquivística brasileira



Módulo 5 - Metodologia para organização das massas documentais acumuladas

a - diagnóstico da situação arquivística;

b - projeto de trabalho.



Módulo 6 - O prédio para instalação do arquivo intermediário

a - especificidades;

b - localização;

c - equipamentos;

d - os depósitos terceirizados de documentos (off site storage).
