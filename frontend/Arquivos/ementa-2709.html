Ementa:

Técnicas de elaboração de projetos para textos acadêmicos. Metodologia para a redação

de projetos de monografia.

Programa:

Unidade I - Identificação, análise e descrição de partes que compõem um projeto para

elaboração de monografia.

1- Elementos pré-textuais.

2- Elementos textuais.

3- Elementos pós-textuais.



Unidade II - Elaboração de projeto harmonizado LP-LSB

1- Composição e redação de projeto nas duas línguas.

2- Elementos pré-textuais.

3- Elementos pós-textuais.

4- Seleção do tema que comporá o projeto.



Unidade III - Metodologia para o desenvolvimento do tema

1- Projeto escrito completo em português.

2- Projeto completo gravado em mídia eletrônica em LSB.

3- Avaliação do projeto harmonizado LSB - Português.
