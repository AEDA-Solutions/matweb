Ementa:

TEORIA: Introdução aos Sistemas a Eventos Discretos, Aplicações para Automação de Processos. Estruturas dos Sistemas de Controle: Processos a Eventos Discretos, Controladores Programáveis. Modelamento das Tarefas de Controle: Descrição do Algoritmo de Controle, Diagramas de Relés. Representação de Sistemas a Eventos Discretos por Redes de Petri. Desenvolvimento do Controle de Sistemas a Eventos Discretos utilizando as Redes de Petri. Metodologias de Projeto de Sistemas de Controle.

LABORATÓRIO: Experiências e Métodos Computacionais no Controle de Sistemas a Eventos Discretos

Programa:

TEORIA



INTRODUÇÃO AOS SISTEMAS A EVENTOS DISCRETOS: Sistemas de Variáveis Contínuas, Sistemas a Eventos Discretos. Histórico dos Sistemas a Eventos Discretos: Controladores por Comutação Elétrica (relés), Controladores por Comutação Eletrônica, Controladores Lógico Programáveis, Controladores Programáveis. Elementos do Controle de um Sistema a Eventos Discretos.



ESTRUTURA DOS SISTEMAS DE CONTROLE: Controle de Sistemas de Variáveis Contínuas, Controle de Sistemas a Eventos Discretos. Características dos Processos Controlados, Controle Local e Distribuído. Representação Matemática dos Controladores Programáveis no espaço de estados discretos. Representação Matemática de um Processo a Eventos Discretos Controlado. Processamento Lógico de Eventos Discretos: lógica combinacional, manutenção de estados, detecção de mudança de estados, processamento temporizado, intertravamento.



REPRESENTAÇÃO DAS TAREFAS DE CONTROLE: Descrição do Algoritmo de Controle. Diagrama de Relés ( LD - ladder diagram): elementos, diagramas e funções de controle, representação matricial. Linguagens Padronizadas: textuais, gráficas e tabulares; elementos; funções; linguagem de máquina e linguagem de controle.



REPRESENTAÇÃO DOS SISTEMAS A EVENTOS DISCRETOS POR REDES DE PETRI: Coceitos Gerais. Redes Condição-Evento. Redes Lugar-Transição. Redes de Marcas Individuais (redes coloridas). Redes de Petri e o Controle de Sistemas a Eventos Discretos.



DESENVOLVIMENTO DO CONTROLE POR REDES DE PETRI: Esquema do Fluxo de Produção (PFS - production flow schema). Gráfico do Fluxo de Marcas (MFG - mark flow graph). Metodologia PFS/MFG



METODOLOGIAS E PROJETO DE SISTEMAS DE CONTROLE: Análise de Necessidades. Definição de Necessidades. Projeto de Sistemas de Controle. Projeto e Programação do Algoritmo de Controle. Desenvolvimento do Programa de Controle e Testes.



LABORATÓRIO

EXPERIÊNCIAS E MÉTODOS COMPUTACIONAIS NO CONTROLE DE SISTEMAS A EVENTOS DISCRETOS
