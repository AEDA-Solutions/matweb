Órgão: EPR - Departamento de Engenharia de Produção
Código: 207462
Denominação: Projeto de Sistemas de Produção 3
Nível: Graduação
Vigência: 1971/2
Pré-req: EPR-205401 Proj de Sistemas de Produção 2
Ementa:

Trabalho em equipe com o objetivo de desenvolver soluções integradas via projetos vinculados a disciplinas obrigatórias ou optativas do Curso de Graduação de Engenharia de Produção.



Programa:

1. Introdução: Visão Geral dos Projetos de Sistemas de Produção (PSPs) Vínculo de Projeto de Sistema de Produção 3 com a disciplina âncora de conteúdo técnico e com os demais PSPs

2. Projeto Preliminar: Elaboração de um plano de projeto. Consolidação de informações (fontes primárias e secundárias). Elaboração da declaração do escopo do projeto e do produto.

3. Projeto Intermediário: Proposição e soluções técnicas para a situação-problema real. Conhecimento de projeto em engenharia. Monitoramento dos objetivos e indicadores do projeto.

4. Projeto final: Entrega final do projeto. Consolidação do projeto preliminar e intermediário. Comparativo planejado x executado em relação ao plano do projeto
