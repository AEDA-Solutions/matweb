Ementa:

Introdução à organização de arquivos de dados e dispositivos de armazenamento externo. Conceitos de sistemas de banco de dados. Modelagem de dados: modelagem conceitual e modelagem relacional. Banco de dados relacional: restrições de integridade, noções de álgebra relacional e cálculo relacional. Linguagem SQL. Projeto de banco de dados relacional: dependências funcionais e formas normais. Processamento de transações.

Programa:



- Dispositivos de armazenamento externo;



- Organização básica de arquivos:

- sequencial;

- direta;

- indexada sequencial.



- Modelagem conceitual

- Banco de dados e os usuários de banco de dados

- Sistemas de banco de dados: conceitos e arquitetura

- Modelagem de dados usando o modelo a entidade-relacionamento



- Modelo Relacional: Conceitos, restrições, linguagens, design e programação

- O modelo de dados relacional e as restrições de um bando de dados relacionais

- A álgebra relacional e o cáculo relacional

- Projeto de banco de dados relacional pelo mapeamento dos modelos

- SQL-99: Definição de esquema, restrições básicas e consultas (Queries)

- Mais SQL: Asserções (assertions), visões (views) e técnicas de programação



- Teoria e Métodos para projetar banco de dados

- Dependência funcional e normalização em um banco de dados relacional

- Método para projeto prático de banco de dados e uso de diagramas UML



- Tópicos em Bancos de Dados

- Segurança e autorização em banco de dados

- Bancos de dados distríbuidos e arquiteturas cliente-servidor

- Banco de dados de objetos e objeto-relacionais
